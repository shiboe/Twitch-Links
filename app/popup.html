Welcome to Twitchlinks 2.0! All links are now listed on the page, accessable by a little icon in the upper right of chat (and likely hiding right behind this wall of text). When new links are present, a notification icon will appear indicating how many are listed. You can still save and hide links like before, but links themselves will now only persist for 3 minutes, keeping the list current to chat. Enjoy, and please create an issue on github if you have one!
